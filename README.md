# Plater
> Make markdown contents lively

## Install

```
npm install plater -g
```

## Usage

1. Install boilerplate
```
npm install plater-bootstrap
```

2. Touch platerfile

```
module.exports = {
    name: 'Title',
    footer: 'Footer',
    boilerplate: 'bootstrap'
};

```

3. Go
```
plater
```
